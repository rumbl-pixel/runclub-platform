💡 How it works

1 Set up your school

Import students via CSV. Auto-generate barcode ID cards for every runner.

2 Scan on run day

Use a Bluetooth barcode scanner or phone camera. Laps auto-log and sync instantly.

3 Track & celebrate

Real-time leaderboards, milestone awards, printable certificates, and parent-visible progress.

What This Is

This is a beta demo

Use it to test the flow, design, and run club workflow with sample data only.

This is not approved for real student data yet

Live use still needs school approval, Supabase Auth, RLS, guardian access, and privacy checks.

This is being prepared for school-specific deployment

Each school should eventually have its own staff access, records, branding, and data boundary.

✅ Platform features

📷
Barcode & QR scanning
Bluetooth scanner or phone camera. Cuts tallying time by up to 90%.
📈
Track setup
Set lap distance, session type, duplicate scan cooldown, and scanner device names.
🏆
Automatic awards
Milestone stickers at 5, 10, 25, 50, 100 & 200 laps. Printable certificates.
📊
Reports & leaderboards
By individual, class, year group. Filter by event, date range, or track.
👥
Parent portal
Parents can view their child's laps, distance, awards, goals, and training.
📄
Roster upload
Bulk import via CSV. Auto-create classes. Print barcode cards (4 sizes).
📅
Jog-a-thon support
Tag laps to fundraising events. Filter reports by event type.
⏱
Timed lap events
Timed mile tracking. Record split times and best lap times per student.
🏠
Coach-managed activity
Staff record approved activity minutes so students cannot add their own totals.
🏞
Multiple programs
Run Club, PE class, recess, Move-a-thon, Brain Booster, Girls Run Club.
🔒
Privacy-first
No student emails. Code-only login. Row-level security per school.
⚡
Offline-ready
Scan without Wi-Fi. Data syncs when connection returns.
