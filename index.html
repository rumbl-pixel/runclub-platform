A digital run club platform for schools – scan student ID codes, track laps, and celebrate progress.

✅ Setup checklist

  1. Later: connect Supabase (for now, demo mode is on).
  2. Decide on classes, houses, and lap distance.
  3. Print ID codes for students.
  4. Use the Admin dashboard to run a demo session.

🏃 What this platform does

  • Rapid lap tallying with barcode scanner input and auto-log.
  • Distance-learning activity tracking and challenge events.
  • School, year, class leaderboards and runner streaks.
  • Automatic awards from lap milestones.
  • Timed tracking support for timed mile events.
  • Marathon-equivalent progress summary for school totals.
  • Local-first backups: students and sessions JSON exports.

📅 Portals

👥 Admin Manage school, students, sessions 📈 Dashboard Scanner, leaderboards, reports 🏃 Student Log in with your code 🔒 Privacy Policy How we handle student data
